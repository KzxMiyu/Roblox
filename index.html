📲 Cài đặt OTA ⬇️ Tải file IPA

📋 Hướng dẫn cài đặt IPA:

  1. Tải file .ipa về máy tính (Windows/Mac)
  2. Cài AltStore, Sideloadly hoặc TrollStore
  3. Kết nối iPhone qua cáp USB
  4. Kéo thả file IPA vào tool → nhập Apple ID
  5. Chờ cài đặt xong → vào Cài đặt → VPN & Quản lý thiết bị
  6. Nhấn Tin cậy chứng chỉ developer
  7. Mở app và sử dụng 🎉
⚠️ Lưu ý: Nếu cài bằng AltStore, app cần được làm mới mỗi 7 ngày. Dùng TrollStore (nếu thiết bị hỗ trợ) để cài vĩnh viễn không cần gia hạn.
